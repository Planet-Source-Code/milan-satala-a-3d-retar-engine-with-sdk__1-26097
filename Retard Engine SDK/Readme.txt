Editor is written in slovak language so don't try to understand it.

 
Game:
 
 W - forward
 S - back
 A - strafe left
 D - strafe right
  
 You can fire at tiger if you hit him he shine red.

Editor:
 
 Moving - hold left button on circle to move veritec, hold elsewhere to move with buffer
 Selecting - right click on buffer
 Scrolling - hold right button and move mouse (fast)

 Editor is ugly and stupid so dont try to create big maps.
